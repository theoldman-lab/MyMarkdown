# 第四章：四元数的深入探讨

## 1. 四元数的定义与代数结构

### 1.1 形式：四元数的标准表示

四元数是复数向四维空间的推广，形如：
$$q = w + xi + yj + zk$$

其中 $w, x, y, z \in \mathbb{R}$，$i, j, k$ 是三个虚数单位。

#### 分量命名

- $w$：**实部**（scalar part）
- $xi + yj + zk$：**虚部**或**向量部**（vector part）

有时也记作：
$$q = (w, \mathbf{v}) \quad \text{其中} \quad \mathbf{v} = (x, y, z)$$

这种表示法将四元数看作一个标量和一个三维向量的组合。

#### 例子

- $q = 1 + 2i + 3j + 4k$：实部为 1，向量部为 $(2, 3, 4)$
- $q = 5 + 0i + 0j + 0k = 5$：纯实数（标量四元数）
- $q = 0 + 1i + 0j + 0k = i$：纯虚四元数

---

### 1.2 基本乘法规则

四元数的乘法由以下基本关系定义：
$$i^2 = j^2 = k^2 = ijk = -1$$

#### 推导乘法表

从 $ijk = -1$ 出发，可以推导出完整的乘法规则：

**步骤 1**：两边右乘 $k$
$$ijk \cdot k = -1 \cdot k \Rightarrow ij(k^2) = -k \Rightarrow ij(-1) = -k \Rightarrow ij = k$$

**步骤 2**：两边左乘 $i$
$$i \cdot ijk = i \cdot (-1) \Rightarrow (i^2)jk = -i \Rightarrow (-1)jk = -i \Rightarrow jk = i$$

**步骤 3**：类似可得 $ki = j$

**完整乘法表**：

| × | 1 | i | j | k |
|---|---|---|---|---|
| **1** | 1 | i | j | k |
| **i** | i | -1 | k | -j |
| **j** | j | -k | -1 | i |
| **k** | k | j | -i | -1 |

#### 记忆技巧：循环图

```
        i
       / \
      /   \
     k ←--- j
    
顺时针：ij = k, jk = i, ki = j
逆时针：ji = -k, kj = -i, ik = -j
```

---

### 1.3 非交换性：四元数的核心特征

#### 反交换关系

四元数乘法**不满足交换律**，具体表现为：
$$ij = k, \quad \text{但} \quad ji = -k$$
$$jk = i, \quad \text{但} \quad kj = -i$$
$$ki = j, \quad \text{但} \quad ik = -j$$

一般地，对于不同的虚数单位：
$$uv = -vu \quad (\text{当 } u, v \in \{i, j, k\} \text{ 且 } u \neq v)$$

#### 例子：验证非交换性

计算 $ij$ 和 $ji$：
- $ij = k$（根据定义）
- $ji = -k$（反交换）

所以 $ij \neq ji$。

#### 实际计算示例

设 $q_1 = 1 + i$，$q_2 = 1 + j$，计算 $q_1 q_2$ 和 $q_2 q_1$：

$$\begin{aligned}
q_1 q_2 &= (1 + i)(1 + j) = 1 + j + i + ij = 1 + i + j + k \\
q_2 q_1 &= (1 + j)(1 + i) = 1 + i + j + ji = 1 + i + j - k
\end{aligned}$$

可见 $q_1 q_2 \neq q_2 q_1$（$k$ 的符号不同）。

---

### 1.4 共轭与范数

#### 四元数的共轭

类似于复数的共轭，四元数 $q = w + xi + yj + zk$ 的**共轭**定义为：
$$q^* = w - xi - yj - zk$$

即保持实部不变，虚部取反。

**性质**：
- $(q^*)^* = q$
- $(pq)^* = q^* p^*$（注意顺序反转！）
- $q + q^* = 2w$（两倍的实部）
- $q - q^* = 2(xi + yj + zk)$（两倍的虚部）

#### 四元数的范数

四元数 $q = w + xi + yj + zk$ 的**范数**（模）定义为：
$$|q| = \sqrt{w^2 + x^2 + y^2 + z^2}$$

这等价于四维空间中的欧几里得长度。

**重要性质**：
$$|q|^2 = q q^* = q^* q = w^2 + x^2 + y^2 + z^2$$

#### 四元数的逆

非零四元数 $q$ 的**逆**为：
$$q^{-1} = \frac{q^*}{|q|^2}$$

验证：
$$q \cdot q^{-1} = q \cdot \frac{q^*}{|q|^2} = \frac{qq^*}{|q|^2} = \frac{|q|^2}{|q|^2} = 1$$

#### 单位四元数

**单位四元数**是范数为 1 的四元数：
$$|q| = 1 \quad \Leftrightarrow \quad q^{-1} = q^*$$

单位四元数在表示旋转时特别重要。

## 2. 四元数表示三维旋转

### 2.1 向量与纯四元数的映射

#### 三维向量的四元数表示

三维空间中的向量 $\mathbf{v} = (x, y, z)$ 可以表示为**纯四元数**（实部为 0）：
$$v = 0 + xi + yj + zk = xi + yj + zk$$

这种映射建立了 $\mathbb{R}^3$ 与纯四元数空间的一一对应。

#### 例子

- 向量 $(1, 0, 0)$ 对应四元数 $i$
- 向量 $(0, 1, 0)$ 对应四元数 $j$
- 向量 $(0, 0, 1)$ 对应四元数 $k$
- 向量 $(1, 2, 3)$ 对应四元数 $i + 2j + 3k$

---

### 2.2 旋转公式

#### 旋转四元数

绕单位轴 $\mathbf{u} = (u_x, u_y, u_z)$ 旋转角度 $\theta$ 的**旋转四元数**定义为：
$$q = \cos\frac{\theta}{2} + (u_x i + u_y j + u_z k)\sin\frac{\theta}{2}$$

简记为：
$$q = \cos\frac{\theta}{2} + \mathbf{u}\sin\frac{\theta}{2}$$

其中 $\mathbf{u}$ 是向量部对应的纯四元数。

**重要**：使用 $\theta/2$ 而不是 $\theta$！

#### 旋转操作

给定向量 $\mathbf{v}$ 对应的纯四元数 $v$，旋转后的向量 $v'$ 由以下公式给出：
$$v' = q v q^{-1}$$

其中 $q$ 是上述旋转四元数。

#### 为什么是 $qvq^{-1}$ 而不是 $qv$？

- $qv$ 的结果一般不是纯四元数（实部不为 0）
- $qvq^{-1}$ 保证了结果仍是纯四元数（实部为 0）
- 这种形式称为**共轭作用**（conjugation）

---

### 2.3 几何机制详解

#### 机制 1：共轭变换保持纯四元数子空间

**定理**：如果 $v$ 是纯四元数，$q$ 是单位四元数，则 $qvq^{-1}$ 也是纯四元数。

**证明思路**：
- 纯四元数的特征是实部为 0
- 共轭变换保持这一性质

这意味着旋转操作将三维向量映射到三维向量，不会"泄漏"到第四维。

#### 机制 2：双重旋转效应

旋转四元数使用 $\theta/2$ 的原因在于**双重旋转**：

$$v' = q v q^{-1}$$

- 左乘 $q$ 产生 $\theta/2$ 的旋转
- 右乘 $q^{-1}$ 产生另一个 $\theta/2$ 的旋转
- 总旋转角度：$\theta/2 + \theta/2 = \theta$

#### 机制 3：双覆盖性质

**重要性质**：$q$ 和 $-q$ 表示**同一个旋转**！

验证：
$$(-q) v (-q)^{-1} = (-q) v (-q^{-1}) = q v q^{-1}$$

这意味着：
- 单位四元数空间（3-球面 $S^3$）**双覆盖**旋转群 $SO(3)$
- 每个旋转对应两个互为相反数的四元数

这与数学中的**$SU(2)$ 群**和**$SO(3)$ 群**的关系密切相关：
$$SU(2) / \{\pm I\} \cong SO(3)$$

---

### 2.4 旋转实例

#### 例子：绕 z 轴旋转 90°

**问题**：将向量 $(1, 0, 0)$ 绕 z 轴逆时针旋转 $90°$。

**解**：

1. **确定旋转轴和角度**
   - 旋转轴：$\mathbf{u} = (0, 0, 1)$（z 轴）
   - 旋转角：$\theta = 90° = \pi/2$

2. **构造旋转四元数**
   $$\begin{aligned}
   q &= \cos\frac{\pi}{4} + k\sin\frac{\pi}{4} \\
   &= \frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}k
   \end{aligned}$$

3. **写出向量的四元数形式**
   $$v = i$$

4. **计算逆四元数**
   $$q^{-1} = \frac{\sqrt{2}}{2} - \frac{\sqrt{2}}{2}k$$

5. **应用旋转公式**
   $$\begin{aligned}
   v' &= q v q^{-1} \\
   &= \left(\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}k\right) i \left(\frac{\sqrt{2}}{2} - \frac{\sqrt{2}}{2}k\right) \\
   &= \left(\frac{\sqrt{2}}{2}i + \frac{\sqrt{2}}{2}ki\right) \left(\frac{\sqrt{2}}{2} - \frac{\sqrt{2}}{2}k\right) \\
   &= \left(\frac{\sqrt{2}}{2}i + \frac{\sqrt{2}}{2}j\right) \left(\frac{\sqrt{2}}{2} - \frac{\sqrt{2}}{2}k\right) \\
   &= \frac{1}{2}(i - ik + j - jk) \\
   &= \frac{1}{2}(i - (-j) + j - i) \\
   &= \frac{1}{2}(i + j + j - i) \\
   &= j
   \end{aligned}$$

6. **解释结果**
   - $v' = j$ 对应向量 $(0, 1, 0)$
   - 这正是 $(1, 0, 0)$ 绕 z 轴旋转 $90°$ 的结果！

## 3. 为何高维数能旋转低维空间

### 3.1 代数嵌入：低维对象嵌入高维代数结构

关键思想是**嵌入**：
- 三维向量 $\mathbb{R}^3$ 嵌入到四维四元数 $\mathbb{H}$ 中
- 嵌入方式：$\mathbf{v} \mapsto 0 + xi + yj + zk$

四元数的额外维度（实部）提供了**操作空间**，使得旋转可以通过代数运算实现。

---

### 3.2 广义欧拉公式

复数中的欧拉公式：
$$e^{i\theta} = \cos\theta + i\sin\theta$$

在四元数中有推广。对于任意满足 $\mathbf{u}^2 = -1$ 的纯单位四元数 $\mathbf{u}$：
$$e^{\mathbf{u}\theta} = \cos\theta + \mathbf{u}\sin\theta$$

**验证**：泰勒展开
$$\begin{aligned}
e^{\mathbf{u}\theta} &= 1 + \mathbf{u}\theta + \frac{(\mathbf{u}\theta)^2}{2!} + \frac{(\mathbf{u}\theta)^3}{3!} + \cdots \\
&= 1 + \mathbf{u}\theta - \frac{\theta^2}{2!} - \mathbf{u}\frac{\theta^3}{3!} + \frac{\theta^4}{4!} + \cdots \\
&= \left(1 - \frac{\theta^2}{2!} + \frac{\theta^4}{4!} - \cdots\right) + \mathbf{u}\left(\theta - \frac{\theta^3}{3!} + \cdots\right) \\
&= \cos\theta + \mathbf{u}\sin\theta
\end{aligned}$$

---

### 3.3 虚数单位的多样性

在复数中，只有两个数满足 $x^2 = -1$：$i$ 和 $-i$。

但在四元数中，有**无穷多个**数满足 $\mathbf{u}^2 = -1$！

**条件**：纯四元数 $\mathbf{u} = xi + yj + zk$ 满足 $\mathbf{u}^2 = -1$ 当且仅当：
$$x^2 + y^2 + z^2 = 1$$

这对应于三维空间中的**单位球面** $S^2$ 上的任意点。

**几何意义**：
- 每个单位向量都可以作为一个"虚数单位"
- 这对应于绕该轴旋转的可能性

---

### 3.4 第四维的作用：参数空间与代数缓冲域

#### 为什么需要第四维？

1. **参数存储**：实部存储了旋转的"标量信息"（$\cos\frac{\theta}{2}$）

2. **代数封闭**：四元数乘法在四维空间中封闭，保证运算的自洽性

3. **避免奇点**：相比欧拉角，四元数表示没有**万向节锁**（Gimbal Lock）问题

4. **插值友好**：四元数球面线性插值（SLERP）在计算机图形学中广泛应用

#### 可视化理解

想象三维空间中的旋转：
- 旋转轴：三维空间中的方向（2 个自由度）
- 旋转角：绕轴转多少（1 个自由度）
- 总共 3 个自由度

四元数有 4 个分量，但单位四元数约束 $|q|=1$ 减少了 1 个自由度，剩下 3 个自由度——正好对应三维旋转！

## 本章小结

### 四元数基础

| 概念 | 公式/定义 |
|------|-----------|
| 标准形式 | $q = w + xi + yj + zk$ |
| 基本关系 | $i^2 = j^2 = k^2 = ijk = -1$ |
| 共轭 | $q^* = w - xi - yj - zk$ |
| 范数 | $|q| = \sqrt{w^2 + x^2 + y^2 + z^2}$ |
| 逆 | $q^{-1} = q^*/|q|^2$ |

### 旋转表示

| 概念 | 公式 |
|------|------|
| 旋转四元数 | $q = \cos\frac{\theta}{2} + \mathbf{u}\sin\frac{\theta}{2}$ |
| 旋转操作 | $v' = qvq^{-1}$ |
| 双覆盖 | $q$ 和 $-q$ 表示同一旋转 |

### 关键洞察

1. **非交换性**是四元数的核心特征
2. **共轭作用** $qvq^{-1}$ 保证旋转后仍是纯四元数
3. **$\theta/2$** 的使用源于双重旋转效应
4. **第四维**提供了旋转操作所需的代数空间
